## ℹ️ 프로젝트 소개

- 원하는 참고문헌을 등록할 수 있습니다.

- 등록한 참고문헌의 4가지 인용 양식(APA, Vancouver, MLA, Chicago)이 자동으로 생성됩니다.

- 문헌별 세부 페이지에 접속하여 하이라이팅, 메모를 할 수 있습니다.
